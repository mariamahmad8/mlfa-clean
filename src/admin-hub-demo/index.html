Reusable, no-code inbox operations.

This demo shows the admin side of the upgraded hub. AI still decides what an email is. Admin settings decide what happens next: folder, forwarding, auto-reply, tags, priority, queue, or full automation. Everything here is static HTML so you can review the product shape before wiring the backend.
